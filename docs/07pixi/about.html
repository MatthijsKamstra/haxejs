About Pixi.js

One of the more popular libraries today is pixi.js.

Super fast HTML 5 2D rendering engine that uses webGL with canvas fallback

To access that sweetness we will be using externs (read more about externs here and here)

The externs written and maintained by Adi Reddy Mora aka adireddy and the current pixi.js version is 4.3.4
